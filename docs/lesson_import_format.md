# 教材インポート形式ガイド
## Gamified Mandala Chart - Lesson Import Format

このドキュメントは、ChatGPT / Gemini / NotebookLM などのAIツールで教材を作成する際に使用するフォーマット仕様です。
このフォーマットに従って出力された内容は、Gamified Mandala Chart の「教材インポート」機能でそのまま取り込むことができます。

---

## 基本構文

```markdown
# [レッスンタイトル]
<!-- level: [必要レベル] -->
<!-- xp: [獲得XP] -->

[レッスンの説明（1〜2文）]

---

[本文：教材の詳細内容]

===

# [次のレッスンタイトル]
...
```

---

## フィールド説明

| フィールド | 必須 | 説明 | 例 |
| :--- | :---: | :--- | :--- |
| `# タイトル` | ✅ | レッスンのタイトル（H1見出し） | `# 目標設定の基本` |
| `<!-- level: N -->` | ❌ | このレッスンを開くために必要なトラのレベル（デフォルト: 1） | `<!-- level: 3 -->` |
| `<!-- xp: N -->` | ❌ | 完了時に獲得できるXP（デフォルト: 50） | `<!-- xp: 100 -->` |
| 説明文 | ✅ | レッスンの概要（1〜2文） | `目標を明確にする方法を学びます。` |
| `---` | ✅ | 説明と本文の区切り線 | `---` |
| 本文 | ✅ | 教材の詳細内容（Markdown対応） | 複数段落、リスト、引用など |
| `===` | ❌ | 複数レッスンの区切り（複数インポート時） | `===` |

---

## 例1：単一レッスンのインポート

```markdown
# 目標設定の基本

<!-- level: 1 -->
<!-- xp: 50 -->

SMARTゴールを使って、達成可能な目標を設定する方法を学びます。

---

## SMARTゴールとは？

SMARTは以下の5つの要素の頭文字です：

- **S**pecific（具体的）
- **M**easurable（測定可能）
- **A**chievable（達成可能）
- **R**elevant（関連性がある）
- **T**ime-bound（期限がある）

## 実践ステップ

1. まず、達成したいことを1文で書き出す
2. 各要素をチェックリストで確認する
3. 必要に応じて目標を修正する

> 💡 **ヒント**: 最初は完璧を目指さず、まず書き出すことが大切です。
```

## 例2：複数レッスンの一括インポート

```markdown
# 習慣化の科学

<!-- level: 1 -->
<!-- xp: 50 -->

習慣がどのように形成されるかを理解し、良い習慣を身につける方法を学びます。

---

習慣は「きっかけ → 行動 → 報酬」というループで形成されます。

このループを理解することで、意図的に良い習慣を作ることができます。

===

# 小さな一歩から始める

<!-- level: 2 -->
<!-- xp: 75 -->

大きな目標を達成するために、まず小さな一歩を踏み出す重要性を学びます。

---

「1日1ページ読む」「1日1分だけ運動する」

このような小さな一歩が、やがて大きな変化につながります。

## なぜ小さく始めるのか？

- 心理的なハードルが低い
- 継続しやすい
- 成功体験が積み重なる

===

# 進捗の可視化

<!-- level: 3 -->
<!-- xp: 100 -->

進捗を見える化することで、モチベーションを維持する方法を学びます。

---

## 進捗トラッカーの活用

- カレンダーにチェックを入れる
- アプリで記録をつける
- このマンダラチャートを活用する

継続した日数が増えるほど、「途切れさせたくない」という気持ちが生まれます。
```

## AIへの指示テンプレート

以下のプロンプトをChatGPT / Gemini / NotebookLM にコピペすることで、インポート可能な形式で教材を生成できます。

```
以下のテーマについて、学習教材を作成してください。

【テーマ】
[ここにテーマを入力]

【出力形式】
以下のMarkdown形式で出力してください：

# [レッスンタイトル]
<!-- level: [1-5の数字] -->
<!-- xp: [50, 75, 100のいずれか] -->

[1〜2文の説明]

---

[詳細な教材内容（見出し、リスト、引用などを活用）]

===

（複数レッスンがある場合は上記を繰り返す）

【条件】
- 初心者にもわかりやすい言葉で書く
- 実践的なステップやヒントを含める
- 1レッスンあたり300〜500文字程度
- レベルは難易度に応じて1〜5で設定
```

---

## 注意事項

1. **エンコーディング**: UTF-8で保存してください
2. **改行**: 段落間は空行を入れてください
3. **特殊文字**: `#`, `---`, `===` は構文として使用されるため、本文中で使う場合は注意してください
4. **画像**: 現在のバージョンでは画像のインポートには対応していません
